## Research

- **[Match Your Loss to Your Cost: Asymmetric Losses and Conformal Capacity Bands for Backbone Traffic Forecasting](https://github.com/SuryanshSS1011/match-loss-to-cost)**\
  Training traffic forecasters on the operator's real provisioning cost instead of RMSE, then calibrating capacity with Adaptive Conformal Inference. Cuts realized cost up to +76% over an MSE baseline at equal accuracy, as a drop-in loss and calibration layer with no new model. *Submitted to CNSM 2026.*
- **[Scheduled Partial-Credit RL for Reliable Code Generation with Small Language Models](https://github.com/SuryanshSS1011/SecureCodeRL)**\
  An RL curriculum that replaces binary pass/fail rewards with graded credit for syntax, execution, and tests to make small models generate more reliable, safer code. Lifts syntax validity to 63% on held-out APPS+ prompts, where binary-reward PPO regresses below the SFT baseline. [*LCTES '26*](https://doi.org/10.1145/3814943.3816167) *(WIP).*
- **[Fixing Performance Bugs Through LLM Explanations](https://github.com/SuryanshSS1011/Performance-Bugs-LLM)**\
  Fine-tuning an LLM to detect, fix, and explain real Java performance bugs over a purpose-built 490-bug dataset from 17 Defects4J projects. Lifts detection accuracy from 67.3% to 83.7%, pairing each prediction with a human-readable explanation of the fix. [*IEEE AITest 2025*](https://ieeexplore.ieee.org/document/11127255).

## Other Recent Projects

- **[edge-inspection-agent](https://github.com/SuryanshSS1011/edge-inspection-agent)**\
  An industrial inspection agent that routes frames between edge and cloud by cost rather than confidence, escalating only uncertain cases to a VLM. One cost inequality replaces hand-tuned thresholds and delivers privacy, offline resilience, and cloud orchestration at once, running 57% cheaper than cloud-only at near-parity accuracy on MVTec AD.
- **[Warren](https://github.com/SuryanshSS1011/Warren)**\
  A home for Wikipedia explorers that turns a rabbit hole into a shareable, interactive map, with LLM-generated connective tissue explaining how each article follows from the last, plus reading and retention tools layered on top. Live at [wikiwarren.vercel.app](https://wikiwarren.vercel.app).
- **[Shift](https://github.com/SuryanshSS1011/Shift)**\
  AI-personalized sustainability actions, surfacing the environmental cost of the AI doing the personalizing alongside each recommendation.
